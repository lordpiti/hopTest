Thanks for taking the time to do our recruitment home exercise

Overview

Create a paged list of cards showing information about recent divisions. Add a text area within each division card allowing a user to save notes against the division.

Instructions

To complete this exercise please modify this solution and do the following:

  • Add components to this application to display a paged list of cards. These cards should display basic information about recent divisions
  • The frontend code in this solution should call the DivisionsController which is configured as an API controller
  • The DivisionsController should get information from the IDivisionInfoService
  • The IDivisionInfoService should get information about divisions from the Divisions API (documented below)
  • In the "Resources" section below two endpoints are documented to help you get information on divisions
  • Displaying basic information on a division is all that is expected. Title, Aye and No vote counts would be a good start.
  • You can use the styles here as inspiration https://votes.parliament.uk but please do not just copy these styles exactly
  • At the bottom of each card you should add a text area and a save button where users can save notes against a division
  • The storage for this has already been implemented for you. You just have to add the endpoints into DivisionsController for your frontend code to call. Please see IDivisionInfoService for existing methods
  • It is expected that if a user saves notes against a division on page 1, then switches to page 2, and then returns to page 1, their notes will be shown
  • Please add tests where appropriate

Resources

  • The divisions API - https://commonsvotes-api.parliament.uk/
  • A call to discover the total number of divisions available - https://commonsvotes-api.parliament.uk/data/divisions.json/searchTotalResults
  • A call to get the first 30 divisions available - https://commonsvotes-api.parliament.uk/data/divisions.json/search?skip=0&take=20
